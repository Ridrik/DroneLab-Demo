# DroneLab - 3D simulator and visualizer tool for drone guidance, navigation, control (GNC)

**Version:** 0.2.2 (Demo)  
**Platform:** Windows 10/11 (64-bit)  
**Developer:** Rodrigo Rosa  
**Date:** October 2025  

DroneLab is a 3D drone visualization and simulation tool developed in **C++ / OpenGL / ImGui**, 
focusing on **Guidance, Navigation, and Control (GNC)**.

## System requirements
System Requirements:
- Windows 10/11 (64-bit)
- OpenGL 3.3 compatible GPU
- ~38 MB free disk space

## Installation
1. Download and run the `DroneLab_Installer_v0.2.2.exe` file.  
2. Follow the on-screen instructions.  
3. Launch DroneLab from the Start Menu or Desktop shortcut.

## File Locations
- Missions: `/missions/` (`.json` configs)  
- Run data: `/runs/` (includes `.json` summaries and `.csv` flight data)
